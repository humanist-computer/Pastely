# Pastely
A pastebin-style application built on Pyhton Flask

### Features
* Syntax highlighting
* Short links

### Start
To start the app, create a virtual env and then:

    python app.py

#### Home
If you open your browser at http://127.0.0.1:5000/ you should see the app home screen:
